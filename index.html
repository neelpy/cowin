CoWIN Vaccine Availability
    This may be useful if you are registered for the vaccine on Co-WIN and are struggling to find a center with an available slot in your district.
    Once you find an available slot, book the slots from the official portal for Co-WIN Registration.


    Features: Saving your state and district, removing unavailable slots, alert on finding an available slot, the rest is same as the original Co-WIN app.
    Implementation: Created using the Co-WIN Public API. This tool is open source, you can find the code here [link].
    Known bugs: The data might be inconsistent with the actual website, because the public API gives slightly older cached data.
    Feedback: If you find any bugs or have any suggestions to improve the tool, you may report them here [link]
